# speech2text
AI 음성 인식 안드로이드 앱

## 개요
경기디지털사회혁신센터 데이터 기술전문가 교육 7회 앱개발 강의를 위해 제작한 예제 안드로이드 앱

## 기술 스택
- Android 21+
- Kotlin 1.4.32
